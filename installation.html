Installation¶

Stable release¶

To install Image Slicer, run this command in your terminal:

$ pip install image_slicer

This is the preferred method to install Image Slicer, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for Image Slicer can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/samdobson/image_slicer

Or download the tarball:

$ curl  -OL https://github.com/samdobson/image_slicer/tarball/master

Once you have a copy of the source, you can install it with:

$ python setup.py install
